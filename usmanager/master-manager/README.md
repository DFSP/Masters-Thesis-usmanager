# Master manager 

Master manager é o módulo principal do sistema de gestão de micro-serviços.  
É um projeto Spring Boot, gerido com maven, que disponibiliza um servidor restful.

### Iniciar

> `mvn package`
>
> `sudo java -jar ./target/master-manager-0.0.1.jar`
